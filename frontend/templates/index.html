About LitReview AI

      

        

          
🏗️

          
Architecture

          
A fully autonomous multi-agent system built with FastAPI. Each agent handles a distinct pipeline stage, communicating through a shared state manager.

        

        

          
🔬

          
Data Sources

          
Papers sourced from Semantic Scholar — 200M+ academic papers. Full text extracted from open-access PDFs and ArXiv preprints.

        

        

          
⚡

          
AI Engine

          
Powered by Groq with Llama 3.3 70B. Ultra-fast inference enables processing 10+ papers in under 3 minutes.

        

        

          
📖

          
Output Quality

          
Produces structured reviews with proper citations, thematic analysis, research gap identification, and export to both Markdown and LaTeX.
